# Calendar AI

A Chrome extension that helps you quickly add events to Google Calendar using natural language or screenshots.

## Features

- **Natural Language Input** — Type events the way you think: "Meet with Sarah tmr 3pm"
- **Screenshot Support** — Paste screenshots of invites or messages
- **Smart Detection** — Understands "tmr", "next Tue", email addresses, and more
- **Google Calendar Sync** — Events appear instantly in your calendar
